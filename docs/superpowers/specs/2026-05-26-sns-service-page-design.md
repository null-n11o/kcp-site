## 1. 概要

X/ThreadsおよびInstagramのSNS運用代行サービスを1ページにまとめた専用ページを新設する。3タブ構成でサービスを切り替えて閲覧でき、ホームのServiceセクションから「詳しく見る」リンクで誘導する。

**URL:** `/services/sns/`

## 3. ホームページへの変更

`src/components/sections/Service.astro` の SNS運用代行の行に「詳しく見る →」リンクを追加。

- リンク先: `/services/sns/`
- `i18n/ja.ts` の `service.items[0]` に `href` フィールドを追加する（`ServiceItem` 型も更新）
- 英語版 (`i18n/en.ts`) にも対応するリンクを追加

---

## 4. 実装方針

### 新規ファイル

| ファイル | 役割 |
|---|---|
| `src/pages/services/sns.astro` | SNS運用代行ページ本体 |

### 変更ファイル

| ファイル | 変更内容 |
|---|---|
| `src/i18n/types.ts` | `ServiceItem` に `href?: string` を追加 |
| `src/i18n/ja.ts` | SNS運用代行アイテムに `href: '/services/sns/'` を追加 |
| `src/i18n/en.ts` | 同上 |
| `src/components/sections/Service.astro` | `href` がある場合に「詳しく見る」リンクを描画 |

### スタイル・JS方針

- 既存の CSS 変数・Tailwind クラスをそのまま使用（`#0e0e0e` 背景、`#7c6aff` アクセント等）
- タブ切り替えは `<script>` ブロックにインラインJS（`querySelectorAll` + `classList`）
- `FadeIn.astro` でスクロールアニメーションを適用
- `PageLayout.astro` を使用（既存ページと同じ構造）

### SEO

- `<title>`: `SNS運用代行 | KCP`
- `<meta description>`: X/Threads・Instagramのそれぞれに対応したSNS運用代行サービスの説明
