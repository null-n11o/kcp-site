# 丸投げIT担当 サービスページ — 設計仕様

## 概要

KCP のメインサービス「丸投げIT担当」の専用ページを新設し、トップページのサービス一覧・ヒーローテキストを更新する。

## 変更スコープ

| 対象ファイル | 変更内容 |
|---|---|
| `src/pages/services/it-support.astro`（新規） | サービス詳細ページ |
| `src/i18n/ja.ts` | サービス一覧に丸投げIT担当を先頭追加 + ヒーローサブテキスト更新 |

英語版（`/en/`）は今回のスコープ外。

---

## 1. サービスページ `/services/it-support/`

### 技術方針

- 既存の `sns.astro` と同じ構成（`PageLayout` + `FadeIn` + `Button`）
- Tailwind ダークテーマ（`#0e0e0e` / `#111` / `#1e1e2e`）
- FAQ は `<details>` / `<summary>` で実装（JS なし）
- タブ切り替えなし（全セクション縦積み）

### セクション構成

#### Hero
- ラベル: `SERVICE`（font-mono accent）
- 見出し: `丸投げIT担当`（h1、既存ページと同スタイル）
- サブテキスト: 「SNSをやりたいけど誰もできない」「AIを使いたいが何から始めればいいかわからない」——月額固定でKCPがあなたの会社のIT・AI担当になります。

#### 解決する課題
- 見出し: `PROBLEM`（font-mono）
- 5項目を `border-b` 区切りのリスト形式で表示
  1. 社内にIT・AIに詳しい人間がいない
  2. SNS・ホームページ・システムの改修を誰に頼めばいいかわからない
  3. 複数の業者に個別発注していて管理が煩雑になっている
  4. 専任のIT担当を採用するほどの規模でも予算でもない
  5. AIを使いたいが導入のハードルが高い

#### サービスの特徴
- 見出し: `FEATURE`（font-mono）
- 3項目を `border-b` 区切りのリスト形式で表示
  1. **プロダクトではなく、人が動くサービスです** — チャットで気軽に相談でき、実際にあらゆるITやAIに関するタスクを完了させるところまでKCPが担います。
  2. **IT・AI特化** — SNS運用・システム改修・LP制作・AI導入支援まで、技術的な業務であれば全て動ける幅広さが強みです。
  3. **チケット制で稼働を可視化** — 月に何をどれだけ依頼したかがチケット単位で明確になります。

#### 料金プラン
- 見出し: `PLAN`（font-mono）
- 3カード（グリッド `sm:grid-cols-3`）、`sns.astro` の Trial/Basic/Plus と同構造

| カード | プラン名 | 月額 | チケット数 |
|---|---|---|---|
| 左 | LIGHT | ¥50,000 | 10チケット |
| 中央（"人気" バッジ） | STANDARD | ¥100,000 | 25チケット |
| 右 | PREMIUM | ¥200,000 | 55チケット |

- カード下に補足: チャット対応無制限・無料、契約条件6か月更新、未使用チケット翌月繰り越しなし

#### チケット消費の目安
- 見出し: `TICKET GUIDE`（font-mono）
- 2カラムテーブル（タスク内容 / 消費チケット数）
- 9行（チャット相談〜新規システム開発）

#### 対応業務範囲
- 見出し: `SCOPE`（font-mono）
- 4カテゴリを 2×2 グリッドカードで表示
  - SNS・コンテンツ（X・Threads・Instagram・TikTok 等）
  - システム・開発（新規開発・改修・ホームページ等）
  - AI活用（AIツール選定・ChatGPT/Claude 導入・自動化等）
  - その他IT全般（SaaS 選定・クラウド設定・IT業者折衝等）

#### 採用との比較
- 見出し: `COMPARISON`（font-mono）
- 6項目の比較テーブル（正社員採用 vs 丸投げIT担当）
  - 月額コスト / 採用コスト / 即戦力性 / 離職リスク / スキルの幅 / 契約柔軟性

#### 向いている会社
- 見出し: `FIT FOR`（font-mono）
- 6項目チェックリスト（`border-b` リスト形式）

#### FAQ
- 見出し: `FAQ`（font-mono）
- 5問を `<details>` / `<summary>` アコーディオンで実装
  1. 未使用チケットは翌月に繰り越せますか？
  2. チケット数が足りなくなった場合はどうなりますか？
  3. 首都圏外でも利用できますか？
  4. 開発そのものも対応できますか？
  5. 契約前に相談できますか？

#### CTA
- 既存ページと同パターン（`border-t-2 border-white` + お問い合わせボタン）
- テキスト: 「現状の課題をお聞きし、どのタスクをKCPに任せると効果的かをご提案します。」
- ボタン: 「まずは無料相談」→ Google Forms URL

## 2. トップページ変更

### `src/i18n/ja.ts`

**`service.items` — 先頭に追加**

```ts
{
  icon: '🧑‍💻',
  title: '丸投げIT担当',
  description: '月額固定でKCPがあなたの会社のIT・AI担当になります。チャット相談から実作業まで一括で引き受けます。',
  href: '/services/it-support/',
},
```

**`hero.subtext` — 更新**

変更前:
> SNS運用代行・受託開発・AI導入支援を通じて、共に変化を起こすパートナーとして伴奏いたします。

変更後:
> 中小企業のIT・AI担当として、月額固定で実作業まで引き受けます。SNS運用・システム改修・AI導入——何でも気軽に頼める「社内のIT担当者」が、今日から持てます。
